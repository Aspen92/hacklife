# Hack Life
  Hack Life is your go to website with the latest life hacks submitted by the creator and followers. It is updated every week with 5 new life hacks that     will help your day to day life become easier.

# Features

### Existing Features
- Navigation Bar

  * Featured at the top of the website under the heading. It's a fully responsive navigation bar and can be used on all devices.
  
  * This section makes it easy for the user to quickly scroll through the website by clicking at "Hacks of the Week!" (This will make the website scroll to     the latest life hacks) and "Contact" (This will make the website scroll to the contact information/social media links and a submit form where they can     submit their own life hacks to be published)
 
<img width="1792" alt="Navigationbar image for readme" src="https://user-images.githubusercontent.com/96183912/193021266-162596ac-6e8a-452f-98a1-3bdb7f49b0d8.png">

- The Landing Page Image

  * The landing page gives the user a clean and easy to navigate feeling.
  
  * The landing page includes a quote to explain to the user exactly what life hacking is about.
  
  <img width="1792" alt="Landing page image for Readme" src="https://user-images.githubusercontent.com/96183912/193021671-6724c8bd-4b00-4a50-bf66-111f0ca7462a.png">

- This Weeks Life Hacks Section

  * The "This Weeks Life Hacks Section" is the section where the user can find the latest life hacks of the current week to implement to their own life.
  
<img width="1792" alt="This weeks life hacks image for Readme" src="https://user-images.githubusercontent.com/96183912/193022439-2194c0a7-cb41-4f17-8fb1-4ac196969e38.png">

- The Footer and submit form Section

  * The footer section includes links to the creators social media and contact information. The icons and contact information will open a new tab to allow     the user to navigate to the different sites easy.
    
  * The footer is valuable to the user as it encourages them to keep connected to the creator via social media. And submit their own life hacks to be           published through the submit form in the bottom right corner of the footer.
  
<img width="1792" alt="Footer submit form section image for Readme" src="https://user-images.githubusercontent.com/96183912/193027748-d93ebcbc-188a-4e17-adef-95f74aa30664.png">
